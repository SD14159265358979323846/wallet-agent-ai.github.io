How it Works

Install a Radix Wallet

1. On Agent — Install SDK npm install wallet-agent-ai-radix npm wallet-agent init
2. Instantiate Contract

Create your PolicyVault on Radix and set Name of agent and spending limits.

3. Deposit

Fund your agent wallet. Always keep some XRD for fees.

4. Whitelist Wallets

Add authorized destinations. The agent can only send to listed wallets.

5. More Info

Click anywhere on this box to open the full guide.
